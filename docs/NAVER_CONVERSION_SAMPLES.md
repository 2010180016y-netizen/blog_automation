# Naver Conversion Template Samples

전환형 템플릿 적용 샘플 2종입니다.

## 1) AFFILIATE 샘플
- 파일: `docs/samples/naver_packages/sample_conversion_affiliate/post.html`
- 특징: 제휴 표기 + 가격/혜택 변동 고지 포함, 추천/비추천·체크리스트·비교표·FAQ(6개) 포함.

## 2) MY_STORE 샘플
- 파일: `docs/samples/naver_packages/sample_conversion_mystore/post.html`
- 특징: 전환형 필수 섹션(추천/비추천·체크리스트·비교표·FAQ 6개) 포함.
